## Styling (CSS Overview)

### Theme
- Dark theme with blue accent colors
- Background: #0f172a
- Accent: #38bdf8
- Cards: #1e293b

---

### Responsive Design
The website is fully responsive using media queries:
- 1024px: Tablet adjustments
- 768px: Mobile layout (stacked sections)
- 480px: Small screen optimization

---

### UI Features
- Hover effects on buttons, skills, project cards, and achievements
- Flexbox used for layout
- Clean and minimal design

---

## Technologies Used

- HTML5
- CSS3 (Flexbox and Media Queries)
